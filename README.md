\# YouTube Analytics Platform



End-to-end YouTube analytics system with:

\- Daily channel tracking

\- Growth analysis

\- Competitor comparison

\- Power BI ready views

\- FastAPI backend



\## Tech Stack

\- Python (FastAPI)

\- MS SQL Server

\- YouTube Data API v3

\- Power BI / Web UI



\## Setup

1\. Create database \& tables using `tables.sql`

2\. Create procedures \& views

3\. Add `.env`

4\. Install dependencies:

&nbsp;  pip install fastapi uvicorn pyodbc google-api-python-client python-dotenv



\## Run

uvicorn main:app --reload



\## Features

\- Channel growth tracking

\- Handle-based channel resolution

\- Daily automation ready
